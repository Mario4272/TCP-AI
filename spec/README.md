# TCP/AI Specification Registry

This directory contains the canonical machine-readable specification for TCP/AI.

## Contents
- `tcpai-v0.3.json`: The single source of truth for v0.3 markers, categories, and corpus schema requirements.

## Usage
Tooling should ingest these JSON files to ensure consistent validation and processing across the project. 

To prevent divergence between these files and the human-readable documentation, see the [Spec Registry Drift Guard](../docs/SPEC_REGISTRY_DRIFT_GUARD.md).
