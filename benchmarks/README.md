# Benchmarks

This directory will contain empirical benchmark results for TCP/AI.

Currently, TCP/AI is in the documentation and specification phase. Future benchmark results will be collected and stored here to empirically validate token reduction percentages and response quality across various LLMs (e.g., GPT-4, Claude 3, Llama 3).

## Submitting Benchmarks
When we are ready to accept benchmarks, please use the `Benchmark Result` issue template to submit your findings. Verified results will be compiled into this directory.
