# TCP/AI Tools

This directory is reserved for future TCP/AI tooling.

Currently, TCP/AI is a documentation-first repository and a human-writable specification. We do not have any executable scripts, packages, or dependencies.

## Planned Tools
In the future, we plan to implement the following tools to support the ecosystem:

1. **Tokenizer Counter**: A utility to accurately compare token counts between natural language and TCP/AI shorthand across different model tokenizers.
2. **Prompt Pair Validator**: A CI tool to validate the structure and syntax of submissions to the prompt corpus.
3. **Benchmark Runner**: An automated script to run corpus pairs against live LLM APIs and measure response quality and token usage.
4. **TCP/AI Linter**: A static analysis tool to enforce TCP/AI syntax conventions in IDEs and editors.
5. **Auto-Compressor (Optional)**: An algorithmic tool to automatically convert natural language into valid TCP/AI shorthand.
