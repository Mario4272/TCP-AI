# TCP/AI v0.2 Benchmark Snapshot

**Date:** 2026-05-16  
**Status:** Milestone Snapshot

## 1. Summary
TCP/AI has reached the v0.2 milestone, featuring a stabilized 100-record benchmark corpus and improved reporting tooling. This snapshot represents our current baseline for token-reduction performance across common conversational and technical tasks.

- **Current Seed Corpus**: `v0.2` (100 records)
- **Historical Baseline**: `v0.1` (36 records)
- **Mean Token Reduction**: ~40.8% (across multiple tokenizers)

## 2. Key Achievements (v0.1 -> v0.2)
- **Corpus Expansion**: Scaled the seed dataset from 36 to 100 high-quality, synthetic prompt pairs.
- **Spec Integrity**: Implemented a canonical machine-readable spec registry (`spec/tcpai-v0.3.json`) and automated drift guards.
- **Tooling Maturity**: Refactored the corpus validator to output category distributions and created a dedicated reporting tool for deterministic results.
- **Benchmarking Standards**: Established multi-tokenizer tiktoken baselines and automated reproducibility checks in CI.
- **Contributor Scaffolding**: Published a formal corpus contribution guide and synthetic data policy.

- **[Specification](SPEC.md)**: The core protocol definition.

## 4. Known Limitations
As of v0.2, the following limitations apply:
- **Semantic Fidelity**: While token reduction is measured, we do not yet have a formal automated rubric for semantic quality preservation.
- **Tokenizer Support**: The current baseline is limited to `tiktoken` (OpenAI) families. Local Hugging Face tokenizer support is planned but not yet implemented.
- **Synthetic Scope**: The corpus is 100% synthetic to ensure safety and privacy; real-world production performance may vary.
- **Human Authoring**: TCP/AI remains a human-writable shorthand; it is not yet compared against machine-automated prompt compressors in this snapshot.

## 5. Safe vs. Unsafe Claims

### Safe Claims
- "TCP/AI v0.2 achieves a measurable token reduction of ~40% on a 100-record synthetic benchmark."
- "The benchmark results are reproducible using committed artifacts and open-source tooling provided in this repo."
- "The v0.3 marker set is stabilized and used consistently across the v0.2 corpus."

### Unsafe Claims
- "TCP/AI preserves 100% of meaning in all cases." (Semantic quality is not yet empirically proven).
- "TCP/AI is the most efficient prompt compression protocol." (Comparative benchmarks against other systems are in progress).
- "TCP/AI has been validated on 10M+ tokens of production data." (Currently limited to the 100-record seed).
