# Security Policy

TCP/AI is currently a documentation-first repository and draft specification. It does not contain executable software, libraries, or scripts at this time. 

## Supported Versions

Currently, there are no executable releases to support.

## Reporting a Vulnerability

If you discover a vulnerability in any future tooling or infrastructure associated with this repository, please do not open a public issue. 

Instead, contact the maintainers directly. (Contact information will be provided once executable software is released). 

For issues related to the specification itself (e.g., edge cases causing prompt injection or jailbreak behavior in specific models), please open a standard issue using the Bug Report template, clearly noting the security implication.
